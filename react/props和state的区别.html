# props和state的区别

  1. props 传递的参数到函数。比如form 传递color 到button组件。
  2. state 是组件的数据。它可以让组件对一些信息保持追踪。比如button按钮记录点击次数 思考？ 父组件通过，props 传递给子组件,但是props的值每次都需要计算。再展示在页面中。重新计算的这个逻辑应该放在哪里？
function Avatar({ person, size }) {
  let thumbnailSize = 's';
  if (size * ratio > 90) {
    thumbnailSize = 'b';
  }
  return (
    <img
      className="avatar"
      src={getImageUrl(person, thumbnailSize)}
      alt={person.name}
      width={size}
      height={size}
    />
  );
}

Props 可以让你将这样的逻辑封装在 Avatar 组件中（并在需要时进行更改），以便每个人都可以使用 Avatar 组件，而不必考虑如何请求和调整图像大小。
