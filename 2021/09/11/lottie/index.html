Lottie - 让动画的实现变得如此简单

动画现状

之前项目对动画往往是通过css自己手写/gif动画/svg动画方式，随着动画效果的日益复杂，一些动画效果存在兼容问题不说，有些也是复杂到难以实现，每次都需要花很长时间在动效的开发上
开发小哥哥和设计小姐姐每次都要碰撞很多次，你懂的

当时调研了两种方案 SVGA 和 Lottie，经过对比两种方案区别和生态，最后选择Lottie，通过官方的提供的lottie-web框架加载json动画文件，提供的方法API很灵活控制动画的播放和速率，团队开发和维护动画的效率终于大大提高了

Lottie VS SVGA

Lottie

http://airbnb.io/lottie/
https://lottiefiles.com/

都是跨平台

SVGA

http://svga.io/

框架 Lottie SVGA
大小 67kb 100kb
生态 airbnb + 社区 YY UED 团队 + 社区
跨平台 web, IOS, Android, and React Native iOS / Android / Flutter / Web / weapp
动画文件 json Protobuf 序列化结构数据格式，序列化的数据体更小，传递效率比xml,json 更高
导出工具 AE Flash / AE
声音 不知道 支持
  • Lottie适合矢量动画
  • SVGA更贴合直播应用场景 SVGA可以很轻松的从远程下载动画资源进行播发，可配置性很高；
