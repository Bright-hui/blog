flex-grow 当容器下元素的(以下都以宽度为例，但不仅限于宽度)没有完全填充的时候，使用flex-grow填充剩余的部分

没有设置flex-grow 的默认情况

100px
150px
200px

容器宽度 600

box1 width = 100

box2 width = 150

box3 width = 200

总 width = 100 + 150 + 200 = 450

父容器 剩余 600-450 = 150 接下来就是怎么瓜分这个剩余宽度

全部设置flex-grow = 1的情况

本身宽度 100px

增加宽度 50px

flex-grow: 1

本身宽度 150px

增加宽度 50px

flex-grow: 1

本身宽度 200px

增加宽度 50px

flex-grow: 1

伸计算公式

宽度 = 本身宽度 + 增加（伸）宽度

增加（伸）宽度 = 每一个元素增加的量等于多余的部分除以所有元素总的flex-grow值然后再乘以当前元素的flex-grow值

box1 增加（伸）宽度 = （150(多余的宽度，要瓜分的宽度) / 3）* 1 = 50

box1 宽度 = 本身宽度 100 + 增加（伸）宽度 50 = 150

box2 宽度 = 本身宽度 150 + 增加（伸）宽度 50 = 200

flex-grow = 1:2:3 的情况

本身宽度 100px

增加宽度 25px

flex-grow: 1

本身宽度 150px

增加宽度 50px

flex-grow: 2

本身宽度 200px

增加宽度 75px

flex-grow: 3

伸计算公式

宽度 = 本身宽度 + 增加（伸）宽度

增加（伸）宽度 = 每一个元素增加的量等于多余的部分除以所有元素总的flex-grow值然后再乘以当前元素的flex-grow值

box1 增加（伸）宽度 = （150 / 6）* 1 = 25

box1 宽度 = 本身宽度 100 + 增加（伸）宽度 50 = 150

box2 增加（伸）宽度 = （150 / 6）* 2 = 50

box2 宽度 = 本身宽度 150 + 增加（伸）宽度 25 = 175

box3 增加（伸）宽度 = （150 / 6）* 3 = 75

box3 宽度 = 本身宽度 200 + 增加（伸）宽度 75 = 275
